## ✨ Overview

> ResearchMind is a multi-agent AI system designed to streamline research—from data collection to evaluation.

It uses **LangChain (LCEL)** and **Mistral** to coordinate multiple agents that perform:
- Web search
- Data extraction
- Insight generation
- Output validation

---

## ⚡ Features

- 🤖 **Multi-Agent Orchestration**  
  Coordinates multiple specialized agents to automate research tasks.

- 🌐 **Real-Time Web Search**  
  Retrieves up-to-date information using Tavily API.

- 🧩 **Modular Pipeline (LCEL)**  
  Built with LangChain Expression Language for scalability and flexibility.

- 📝 **Automated Report Generation**  
  Converts raw data into structured research outputs.

- ✅ **Result Evaluation**  
  Validates generated insights for relevance and quality.

- 📊 **Interactive UI**  
  Streamlit-based interface with dynamic rendering and state management.

## 🚀 Installation

1. Clone the repository
```bash
git clone <your-repo-link>
cd researchmind
```

2. Create virtual environment and activate it (recommended)
 
3. Install Dependencies
```bash
pip install -r requirements.txt
```

4. Run the application
```bash
streamlit run app.py
```
